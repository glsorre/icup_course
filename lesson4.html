Lists


A list is an ordered set of values, where each value is identified by an index.
Each of those values is called an element of the list. 


              # a list of strings
              expedia_brands = ['Hotels.com', 'EAN', 'Expedia'] 
              # a list of numbers
              exam_grades = [40, 55, 99, 73]
              # a list containing mixed types
              a_bit_of_everything = ['hello!', 2, True, 0.535]
              # a list of lists
              words = [['hello', 'yes', 'no'], ['hola', 'si', 'no']]

You can pick out items from a list by using their index.
The index starts from 0! 

                print(expedia_brands[0])  # 'Hotels.com'
                print(expedia_brands[2])  # 'Expedia'

                print(exam_grades[0])  # 40
                print(exam_grades[-1])  # 73
                print(exam_grades[-2])  # 55

Why use lists?

A list acts as a single variable to store lots of data

Here are some basic list tricks: 


              # Start with an empty list
              sentence = ['Programming', 'is']
              # Add an item to the end
              sentence.append('fun')

              print(sentence)

Why use lists?

We can concatenate two lists 

            # Create two lists
            sentence_start = ['Programming', 'is']
            sentence_end = ['really', 'fun']
            
            # Join the two lists together using the + operator
            sentence = sentence_start + sentence_end
            print(sentence)
It is useful to know the number of items in a list too: 

            # Create a list
            sentence = ['Programming', 'is', 'really', 'fun']
            
            length = len(sentence)
            print(length) # 4

Why use lists?

Sometimes it's useful to find if an element exists in a given list. 

            expedia_brands = ['Hotels.com', 'EAN', 'Expedia']

            if 'Booking.com' in expedia_brands:
              print('Something must be wrong!')

            if 'Hotels.com' in expedia_brands:
              print('Yay!')
There are shortcuts of creating lists of numbers: 

            # Create a list containing the numbers from 1 to 5
            numbers = range(1, 6) #note that the upper end is non-inclusive

Why use lists?

We can sort lists too 

            random_numbers = [5, 3, 4, 6, 1, 2]
            sorted_numbers = sorted(random_numbers)
            print(sorted_numbers) # displays [1, 2, 3, 4, 5, 6]
Or just take slices of a list 

            letters = ['a', 'b', 'c', 'd', 'e', 'f']
            letters[1:3] # results in ['b', 'c']
            letters[2:3] # results in ['c']
