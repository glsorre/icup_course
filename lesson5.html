Lists

Lists are ordered collections of data 


                expedia_brands = ['Hotels.com', 'EAN', 'Expedia']

                exam_grades = [40, 55, 99, 73]

You can pick out items from a list by using the position

The position starts from 0! 

                print(expedia_brands[0])  # 'Hotels.com'
                print(string_list[2])  # 'Expedia'

                print(exam_grades[0])  # 40
                print(exam_grades[-1])  # 73

Why use lists?

A list acts as a single variable to store lots of data

Here are some basic list tricks: 


              # Start with an empty list
              sentence = []
              # Add an item to the end
              sentence.append('Programming')
              # Add two lists together (note the reuse of the variable)
              sentence = ['terrible...', 'is', 'Python', 'in'] + sentence
              # Reverses the order - note the empty brackets!
              sentence.reverse()
              # Remove a matching item
              sentence.remove('terrible...')
              sentence.append('great!')

              print(sentence)

Fixing our code before it breaks

What happens if we omit any of the punctuation from 'terrible...' in the last slide?


Our code breaks!

Fixing our code before it breaks

We can stop this from happening by checking the element exists first 


                sentence = ['Programming', 'in', 'Python', 'is', 'terrible...']

                word_to_remove = 'terrible...'

                if word_to_remove in sentence:
                    sentence.remove(word_to_remove)
                    sentence.append('great!')
                else:
                    sentence.append('...NOT!')

                print(sentence)

Looping

Since lists contain lots of items, we can use while to loop through them 


                sentence = ['Programming', 'in', 'Python', 'is', 'great!']

                formatted = ''  # an empty string
                position = 0
                
                while position < len(sentence):
                    formatted = formatted + ' ' + sentence[position]
                    position = position + 1

                print(formatted)

For loops

However, it's a pain to keep track of the list position!

This is what for loops do: 


                some_numbers = [0, 28374, 39, 4]

                total = 0

                for number in some_numbers:
                    total = total + number

                print(total)
The number variable is recreated every round of the loop, with the value of the next item in the list
