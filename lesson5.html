Looping

Since lists contain lots of items, we can use while to loop through them 


                sentence = ['Programming', 'in', 'Python', 'is', 'great!']

                formatted = ''  # an empty string
                position = 0
                
                while position < len(sentence):
                    formatted = formatted + ' ' + sentence[position]
                    position = position + 1

                print(formatted)

For loops

However, it's a pain to keep track of the list position!

This is what for loops do: 


                some_numbers = [0, 28374, 39, 4]

                total = 0

                for number in some_numbers:
                    total = total + number

                print(total)
The number variable is recreated every round of the loop, with the value of the next item in the list
