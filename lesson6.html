Dictionaries

So far we have seen lists - collections of data, where each element has an index. This index is a number.

Dictionaries are similar to lists, except the index of each element can be a string too! 


                eng_to_sp = {} # create an empty dictionary

                eng_to_sp['one'] = 'uno' # add information to it
                eng_to_sp['two'] = 'dos'

                print(eng_to_sp)

Dictionaries

The indices are called keys.
Each key has a value associated with it.
So the elements are called key-value pairs.


                # We can define the dictionary in one line too
                eng_to_sp = {'one': 'uno', 'two': 'dos', 'three': 'tres'}

                print(eng_to_sp['one']) # to print a value from the dictionary

Dictionary operations

We can change the value for a given key or delete key-value pairs! 


                inventory = {'apples': 320, 'pears': 228, 'bananas': 420}

                # We can add oranges to our inventory
                inventory['oranges'] = 317
                # If someone buys all the pears, we can reassign their value
                inventory['pears'] = 0

                # Or if we expect to stock no more pears, we can delete the value!
                del inventory['pears']

                # Let's print what is left in the dictionary
                print(inventory)

                # We can find the number of key-value pairs in the dictionary
                len(inventory) # just like for lists

Dictionary methods

We can get lists of all keys or all values. 


                inventory = {'apples': 320, 'pears': 228, 'bananas': 420}

                print(inventory.keys()) # will print ['apples', 'pears', 'bananas']
                print(inventory.values()) # will print [320, 228, 420]

Or we can check if a dictionary contains a given key, just like in lists. 


                inventory = {'apples': 320, 'pears': 228, 'bananas': 420}

                'apples' in inventory # will return True
                'oranges' in inventory # will return False
